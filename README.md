# CO2 Emissions from the Combustion of Liquid and Gaseous Fuels in Austria

This project is part of the bachelor thesis from Elijah Stängl under the supervision from Johannes Schmidt.

## Project

The goal of this project is to calculate up-to-date CO2 emissions from fuel combustion in Austria. The results should mimic official emission data published in Austria's National Inventory Document.

Data from Eurostat, E-Control, Austria's Common Reporting Table 2026 as well as Austria's National Inventory Document 2026 is used.
